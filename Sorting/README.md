# Сортировка пузырьком

## Сортировка пузырьком представляет собой простой алгоритм сортировки, который проходит по списку несколько раз, сравнивая два соседних элемента и меняя их местами, если они находятся в неправильном порядке. Этот процесс повторяется до тех пор, пока весь список не будет отсортирован.

## Теория:
1. Проход по списку начинается с начала. Два соседних элемента сравниваются.

2. Если порядок элементов неправильный (меньший элемент следует за большим), то они меняются местами.

3. Процесс повторяется для каждой пары элементов в списке.

4. После первого прохода, наибольший элемент "всплывает" на верх списка.

5. Шаги 1-4 повторяются для подсписка, исключая уже отсортированные элементы.

6. Процесс повторяется до тех пор, пока весь список не будет отсортирован.

## Пример работы:
Исходный массив: [64, 34, 25, 12, 22, 11, 90]

После первого прохода: [34, 25, 12, 22, 11, 64, 90]

После второго прохода: [25, 12, 22, 11, 34, 64, 90]

И так далее, пока массив полностью не отсортирован.

Сортировка пузырьком неэффективна для больших списков, но она проста в понимании и реализации.


# Сортировка выбором

## Описание:
Сортировка выбором — это простой алгоритм сортировки, который применяется к массивам данных. Он основан на принципе выбора элемента с наименьшим (или наибольшим) значением из неотсортированной части массива и перемещения его в начало (или конец) отсортированной части.

## Теория:
* Начало процесса:
На каждом шаге выбирается элемент с наименьшим значением из неотсортированной части массива.
* Обмен элементов:
Выбранный элемент меняется местами с первым элементом неотсортированной части массива.
* Сдвиг границы:
Граница между отсортированной и неотсортированной частями смещается, и процесс повторяется для оставшихся элементов.
* Итог:
После выполнения всех шагов весь массив становится отсортированным.
